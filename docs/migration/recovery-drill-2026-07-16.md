# 2026-07-16 人工切换演练记录

## 范围

本次演练验证“发现主站持续不可用后，人工核对 Pages 温备并给出备用入口，主站恢复后再撤销临时通知”的技术链路。演练不人为下线 Vercel，不修改 DNS，也不向外部渠道发送测试通知。

当前没有自定义域名，因此真正发生故障时的接管动作是发布并传播以下备用入口，而不是透明切换：

```text
主站当前访问异常，临时入口：
https://original4422.github.io/original-blog/
```
